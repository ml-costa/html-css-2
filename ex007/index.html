Principais formatações:

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <B> (não semántica).

Nessa frase temos um termos em destaque com <strong> (semática).

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (não semántica).

Nessa frase temos um termo em ênfase com <em> (semántica).

Texto marcado

Podemos também criar um texto marcado usando a tag <Mark> (semantico).

Nesse texto, temos outro texto marcado no final.

Texto Grande e Pequeno

Nessa frase temos um texto grande usando a tag <big> (não semántica).

Nessa frase temos um texto pequeno usando a tag <small> (não semántica).

Texto Deletado

Nessa frase temos um texto excluído para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Nessa frase temos um texto incluído para indicar que ele foi adicionado depois.
